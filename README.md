# 📊 Data Professional Survey Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built using a survey dataset collected from data professionals across different countries, job roles, and experience levels.

The dashboard provides valuable insights into salary trends, programming language preferences, work-life balance, job satisfaction, and the challenges faced when entering the data industry.

## Project Objectives

* Analyze survey responses from data professionals.
* Identify salary trends across different job roles.
* Understand popular programming language preferences.
* Measure employee satisfaction with salary and work-life balance.
* Explore the perceived difficulty of breaking into the data field.
* Present insights through interactive visualizations and KPIs.

## Dashboard Features

### Country-wise Survey Distribution

Visualizes the number of survey participants from different countries.

### Average Salary by Job Title

Compares average salaries across various data-related professions.

### Favorite Programming Language Analysis

Highlights the most preferred programming languages among respondents.

### Work-Life Balance Score

Displays overall satisfaction with work-life balance using KPI gauges.

### Salary Satisfaction Score

Measures respondents' satisfaction with their current salary levels.

### Difficulty Breaking into Data

Analyzes how respondents perceive the challenge of entering the data industry.

### Key Performance Indicators (KPIs)

* Average Age of Respondents
* Total Survey Participants
* Work-Life Balance Rating
* Salary Satisfaction Rating

---

## Key Insights

* Data Scientists reported the highest average salaries among surveyed roles.
* Python emerged as the most preferred programming language.
* Respondents generally reported higher satisfaction with work-life balance than with salary.
* A significant portion of participants indicated that entering the data industry is moderately difficult.
* Survey responses were collected from professionals across multiple countries and job categories.
